星界记忆序列 - 界面设计方案
=======================

1. 设计理念
-----------------
整体风格：
- 简约而富有科技感
- 强调空灵和神性的视觉效果
- 使用渐变蓝色营造星界氛围
- 半透明效果增强层次感

2. 色彩方案
-----------------
主色调：
- 背景基色：columbia-blue (#CFE8EF)
- 主要界面元素：columbia-blue-2 (#C6DBF0)
- 交互元素：columbia-blue-3 (#AED1E6)
- 重点突出：powder-blue (#A0C4E2)
- 特效点缀：sky-blue (#85C7DE)

渐变应用：
- 主界面背景：$gradient-bottom-right
- 卡牌边框：$gradient-radial
- 技能特效：$gradient-top-right

3. 界面布局
-----------------
3.1 主游戏界面
┌─────────────────────────────────┐
│     回合信息 & 行动条          │
├─────────────┬───────────────────┤
│             │                   │
│   棋盘区域  │   角色信息面板   │
│             │                   │
│             │   技能操作区      │
├─────────────┴───────────────────┤
│        系统信息 & 控制栏        │
└─────────────────────────────────┘

3.2 棋盘设计
- 格子大小：80x80像素
- 网格线：1px solid rgba(174, 209, 230, 0.3)
- 可移动范围：半透明 powder-blue 叠加
- 攻击范围：半透明 sky-blue 叠加

3.3 角色立绘展示
- 大小：120x160像素
- 边框：渐变光效
- 悬浮效果：轻微上浮动画
- 受伤效果：红色闪烁

4. 交互设计
-----------------
4.1 移动操作
- 点击角色：显示可移动范围
- hover效果：格子高亮
- 移动动画：平滑过渡

4.2 技能释放
- 技能选择：弧形菜单
- 范围预览：动态显示
- 释放动画：粒子特效

4.3 状态显示
- 生命值：渐变进度条
- buff/debuff：图标+计时器
- 技能CD：环形进度

5. 动画效果
-----------------
5.1 过场动画
- 关卡切换：淡入淡出
- 回合交替：旋转切换

5.2 战斗动画
- 攻击：闪光 + 位移
- 技能：粒子系统
- 受伤：抖动 + 数字跳动

5.3 UI动画
- 按钮：hover缩放
- 面板：滑入滑出
- 提示：渐显渐隐

6. 特殊效果
-----------------
- 光晕效果：角色周围
- 水晶质感：技能图标
- 星光点缀：背景装饰
- 半透明玻璃态：信息面板

7. 响应式设计
-----------------
- 最小分辨率：1280x720
- 棋盘区域：自适应缩放
- UI元素：弹性布局
- 移动端：横屏设计，提示横屏

8. 可访问性
-----------------
- 高对比度模式
- 可调整字体大小
- 操作提示
- 音效反馈
